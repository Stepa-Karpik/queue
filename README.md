# Queue Bot

Проект Telegram-бота для определения очередности сдачи лабораторных и практических работ.

## Быстрый старт
1. Скопируйте `.env.example` в `.env` и заполните `BOT_TOKEN`.
2. Запустите через Docker:
   ```bash
   docker compose up --build
   ```

## Команды
- `/start` — регистрация
- `/list` — импорт списка группы (только староста)
- `/add_subject` — добавить дисциплину (староста)
- `/remove_subject` — удалить дисциплину (староста)
- `/add_work` — добавить работу (староста)
- `/remove_work` — удалить работу (староста)

## Роли
Роль `starosta` назначается в БД вручную (поле `users.role`).

## Приоритет сдачи
Формула реализована в `bot/services/priority.py` и зависит от:
- доли оставшихся работ
- бонуса за 1-2 оставшиеся работы
- фактора неактивности
- штрафа за низкую активность
- модификатора среднего балла
